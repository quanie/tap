Submitting bugs and suggestions

Please use GitHub issues to notify us of a bug or to request a new feature. Before adding a new request, please search the existing issues to see if there is one the same or similar to yours. If so, add a reaction (like :+1: or :-1:) to the issue and post any additional relevant comments that will be helpful.
